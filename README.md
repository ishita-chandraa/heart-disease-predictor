# ❤️ Heart Disease Predictor

A Machine Learning web application that predicts the likelihood of heart disease using **Logistic Regression**.

Built with **Python, Scikit-learn, Streamlit, and Joblib**.

## 📌 Features

- Predicts the probability of heart disease
- User-friendly web interface built with Streamlit
- Real-time risk prediction
- Displays prediction confidence
- Shows model evaluation metrics
- Educational medical tool

---

## 📊 Model Information

Algorithm:
- Logistic Regression

Dataset:
- Cleveland Heart Disease Dataset

## ⚙️ Installation

Clone the repository

```bash
git clone <repository-url>
```

Move into the project folder

```bash
cd heart-disease-predictor
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
